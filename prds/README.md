# Product Requirements Documents (PRDs)

This directory contains Product Requirements Documents for the project.

## Imported PRDs

The following PRDs have been imported from CodeSpring:

1. **Notes Dashboard** (`notes-dashboard.md`)
   - Central hub for viewing, searching, filtering, and navigating notes
   - Includes pagination, real-time updates, and quick actions

2. **Rich Text Editor** (`rich-text-editor.md`)
   - WYSIWYG editor with Tiptap integration
   - Autosave functionality and formatting options

3. **Category Management** (`category-management.md`)
   - Create, rename, and delete categories
   - Assign notes to categories and filter by category

4. **Database & CRUD Operations** (`database-crud-operations.md`)
   - Core backend operations for notes
   - Secure Create, Read, Update, Delete with real-time updates

## Importing Additional PRDs from CodeSpring

To import additional PRDs from CodeSpring, the MCP server is already configured. You can use the CodeSpring MCP integration to fetch new PRDs.

## PRD Structure

Each PRD includes:
- Title and description
- User stories and acceptance criteria
- Technical implementation details
- Database schema
- API endpoints / Server Actions
- Component structure
- Dependencies & integrations
- Implementation steps
- Edge cases & error handling
- Testing approach
